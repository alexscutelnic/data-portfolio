<h1>Supplier Map Power BI Report</h1>

Transform raw supplier and postcode data into an interactive geospatial sourcing tool.
This report allows users to instantly identify nearby suppliers based on distance, service type, and region, helping teams make faster, more strategic operational decisions.

<hr>

<h2>Purpose</h2>

Identify suppliers within a chosen distance from any postcode.
Filter suppliers by service, region, and distance range.
Support data-driven sourcing and logistics decisions.
Provide an intuitive, map-based interface for quick insights.
